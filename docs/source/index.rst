===================================================
Welcome to Team birdsarentreal's REGS Documentation
===================================================

.. epigraph::

   "An idiot admires complexity, a genius admires simplicity."

   -- Terry A. Davis

Welcome to the official developer documentation for the **REGS (Registration System)**, built by Team **birdsarentreal** for CSCI 2524W.

While our team name might suggest a conspiracy about government drone surveillance, this system is actually a robust university registration platform. This documentation covers the backend architecture, database schema, and API routes used to manage the course catalog, student enrollments, academic plans, and user authentication.
